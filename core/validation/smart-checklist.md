# چک‌لیست اعتبارسنجی SMART

## معیارها
- Specific (مشخص بودن): ✔️ / ❌
- Measurable (قابل اندازه‌گیری): ✔️ / ❌
- Achievable (قابل دستیابی): ✔️ / ❌
- Relevant (مرتبط بودن): ✔️ / ❌
- Time-bound (زمان‌مند بودن): ✔️ / ❌

## امتیازدهی
هر معیار: 0 تا 20 امتیاز  
جمع کل: 100 امتیاز

## نتیجه
- ≥ 80: معتبر
- 60–79: نیاز به اصلاح
- < 60: نامعتبر
